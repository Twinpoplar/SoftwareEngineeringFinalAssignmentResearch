## 一、系统名称
在线考试系统

---

## 二、技术选型

### 核心框架
- React 18.2 + TypeScript 5.3
- 构建工具：Vite 5.0

### 状态管理
- 客户端状态：Zustand 4.4
- 服务端状态：TanStack Query (React Query) 5.0

### UI 方案
- 管理后台：Ant Design 5.11
- 考试界面：TailwindCSS 3.3（自定义设计）
- 图标库：Lucide React

### 其他依赖
- 路由：React Router 6.20
- HTTP 客户端：Axios + 自定义拦截器
- 实时通信：Socket.io-client 4.7
- 富文本编辑：ReactQuill 2.0（简答题）
- 表单处理：React Hook Form 7.48 + Zod 验证
- 日期处理：Dayjs 1.11
- 开发工具：ESLint 8 + Prettier 3 + Husky 预提交钩子

## 四、界面开发要求

### 登录界面
- 功能：
  - 角色选择（学生/教师/管理员）
  - 账号密码登录
  - 记住密码（管理员不可选）
  - 备注:
    - 管理员账号为admin
    - 管理员密码为TestAdmin123
    - 注册页面不可选择注册为管理员
- 技术：React Hook Form + Zod 验证，密码 bcrypt 前端预加密

### 考试列表界面（学生端）
- 功能：
  - 考试状态标签（未开始/进行中/已完成）
  - 倒计时显示
  - 进入考试按钮状态控制
- 技术：React Query 缓存，倒计时使用 requestAnimationFrame 优化

### 在线答题界面（学生端）
- 布局：左侧题目区(70%) + 右侧答题卡(30%)
- 功能：
  - 单选/多选/判断/简答题型渲染
  - 上一题/下一题/答题卡跳转
  - 题目状态标记（每道题有独立“未做/已做”，且所有题目有统一鸟瞰图标记）
  - 简答题富文本编辑 + 图片上传
  - 实时倒计时（顶部悬浮，默认可移动浮窗，选择最小化则吸附在页面左上角）
  - 自动保存提示
- 技术：
  - 富文本：ReactQuill 自定义图片上传模块
  - 防作弊：Page Visibility API 切屏检测，Fullscreen API 全屏锁定
  - 自动保存：防抖保存(30 秒) + 离开页面提示
  - 快捷键：方向键切换，数字键跳转，Ctrl+S 保存

### 试卷提交界面（学生端）
- 流程：二次确认弹窗 → 未作答提示 → 提交中 Loading → 结果反馈
- 防误触：提交按钮需长按 3 秒或输入确认码

### 成绩查询界面（学生端）
- 功能：
  - 分数展示
  - 排名显示
  - 错题回顾（仅显示个人答案）
  - 教师评语
- 图表：使用 Recharts 展示知识点掌握雷达图

### 题库管理界面（教师端）
- 功能：
  - 题目检索（多条件筛选）
  - 批量导入（Excel 模板）
  - 批量导出
  - 题目预览
  - 审核状态
- 表格：Ant Design Table 虚拟滚动（大数据量）

### 考试管理界面（教师端）
- 功能：
  - 发布考试（挑选试题 + 考试时间）
  - 成绩评定（参加考试的学生的成绩分布、排名）

## 五、交互流程实现

- 考试开始校验：进入答题页前校验：身份 Token 有效 + 考试时间窗口 + 无进行中的其他考试
- 答题切换：本地状态管理 + 自动保存到 Redis（通过 API）
- 倒计时预警：剩余 15 分钟黄色提醒，剩余 5 分钟红色闪烁 + 声音提示
- 提交确认：检测未作答题目 → 弹窗列表展示 → 确认提交 → 防重复提交 Token
- 强制交卷：Socket.io 监听 server 强制交卷事件，自动提交并跳转

## 六、状态管理规范

### Zustand Store 划分
- authStore：用户信息、权限、Token
- examStore：当前考试状态、答题进度、倒计时
- uiStore：全局 Loading、消息提示、模态框状态

### React Query 配置
- staleTime：考试列表 5 分钟，题目数据 10 分钟
- cacheTime：成绩数据 30 分钟
- refetchOnWindowFocus：false

---

## 七、API 接口规范

- 基础配置：Axios 实例封装，统一错误处理，Token 自动刷新
- 请求拦截：添加 Timestamp 签名防重放
- 响应拦截：统一处理 401/403/500 错误码
- 文件上传：分片上传（大图片）+ 上传进度显示

---

## 八、性能优化要求

- 代码分割：按路由懒加载，考试界面独立 chunk
- 资源优化：图片懒加载，富文本编辑器动态导入
- 渲染优化：React.memo 优化选项列表，useMemo 缓存计算
- 网络优化：答题自动保存防抖(5 秒)，失败重试 3 次
